# User-Centric Decision Intelligence System

## Transforming Raw Data into Actionable Insights

A Flask-based Decision Intelligence platform that enables users to upload datasets, perform automated and manual analysis, generate visualizations, and receive actionable insights for data-driven decision-making.

This project bridges the gap between raw data and intelligent actions by integrating analytics, visualization, recommendation generation, and history tracking into a single user-friendly platform.

---

## Project Overview

Organizations and individuals generate large amounts of data every day. However, transforming that data into meaningful decisions remains a challenge.

The User-Centric Decision Intelligence System addresses this challenge by providing an interactive platform where users can:

* Upload datasets
* Analyze data automatically or manually
* Generate visual reports
* Receive insights and recommendations
* Track historical analyses

The system demonstrates the practical application of Decision Intelligence in supporting smarter and faster decision-making.
